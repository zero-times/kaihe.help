## 一句话结论

提醒模板的作用不是“解释清楚”，而是让家人和朋友先别乱回、别乱转、别补充更多信息。

## 发给家人的提醒模板

```text
我这边正在处理一件网上公开个人信息的事情。

如果你收到陌生电话、短信、私信，先不要回应、不要回拨、不要解释，也不要把截图转发给其他人。

如果对方提到我的姓名、手机号、学校、住址、工作或家庭情况，请把时间、号码、截图发给我，我统一处理。
```

## 发给朋友 / 同学的提醒模板

```text
最近有人在网上扩散和我有关的隐私信息。

如果你看到相关内容，先不要评论、转发、截图扩散，也不要替我公开解释。

如果你看到了原帖、搬运内容或相关账号，直接把链接和截图发给我就行。
```

## 发给同住人或伴侣的提醒模板

```text
现在最重要的是先收紧公开入口，不要让更多现实信息被拼出来。

麻烦先帮我一起看一下：定位、相册、社交主页、外卖昵称、二手平台、公开简历这些入口里，有没有带住址、单位、学校或家人信息的内容。

如果有陌生人联系你，先别多说，只把时间、号码和内容记下来发给我。
```

## 陌生联系统一回复模板

如果你必须回复，可以尽量短：

```text
请不要继续传播涉及个人隐私的信息。相关内容我已留存处理。
```

能不回复时，优先选择不回复，先留证。

## 使用时注意

1. 只发给需要知道的人，不要把事情一次扩散到更大的群。
2. 模板里的“我的姓名、手机号、住址、学校”等内容，按你实际风险替换，不要多写。
3. 让家人和朋友知道“把信息发回给你”，比“他们自己去处理”更重要。
4. 如果已经出现威胁、堵门、跟踪、骚扰家人，不要只靠提醒模板，尽快升级处理。

## 哪些话先别发

1. “事情很复杂，我从头给你讲。”
2. “你帮我去评论区解释一下。”
3. “我把截图发群里，大家都看看。”
4. “如果对方再来找你，你就替我骂回去。”
