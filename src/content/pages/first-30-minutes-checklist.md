## 第 0-5 分钟

- [ ] 先深呼吸，不要立刻公开回应。
- [ ] 截图当前页面全貌、评论区、发布时间、账号主页。
- [ ] 复制原始链接、评论链接、账号主页链接。
- [ ] 录一段能看到页面滚动和账号名的短视频。
- [ ] 写下你第一次发现的时间和平台。

## 第 5-15 分钟

- [ ] 判断到底公开了什么：手机号、住址、学校、单位、家人信息，还是聊天记录、照片。
- [ ] 搜同平台和其他平台，看是否已有搬运、转发或评论区继续补充信息。
- [ ] 先改高风险账号隐私设置，关掉能继续暴露的信息入口。
- [ ] 打开骚扰拦截、陌生号码静音、短信过滤等功能。
- [ ] 如果家人或朋友可能被联系，先通知最关键的人，不要大范围转发截图。

## 第 15-30 分钟

- [ ] 把最核心的证据整理成一个文件夹或备忘录。
- [ ] 先做平台内举报和申诉，尤其是原帖、评论区、搬运内容。
- [ ] 记下对方账号名、链接、扩散节点和你已经做过的动作。
- [ ] 判断是否已经出现线下风险、家人风险、未成年人风险或持续骚扰。
- [ ] 如果有明显人身威胁、住址学校泄露、家人被骚扰，尽快准备报警材料。

## 今晚前补做

- [ ] 补看自己和家人最容易暴露的入口，例如旧社交账号、二手平台、外卖昵称、公开简历。
- [ ] 统一一段给家人或朋友的提醒话术。
- [ ] 记录后续新增骚扰电话、短信、私信或搬运链接。
- [ ] 把“发现时间 - 公开内容 - 扩散节点 - 已做处理”整理成简短时间线。

## 哪些先别做

- [ ] 先别发长文解释。
- [ ] 先别删除最关键证据。
- [ ] 先别把完整截图发到多个群里。
- [ ] 先别只盯着一条帖，忽略评论区和搬运号。
- [ ] 先别以为删帖就等于风险结束。
