## 一句话结论

判断顺序看的是**风险等级**，不是看你更生气还是更想立刻解决。没有明显线下风险时，通常先平台举报和留证；一旦出现人身威胁、住址、家人或未成年人信息，就不要拖。

## 先用这张表判断

| 场景 | 更优先的动作 |
| --- | --- |
| 主要是公开辱骂、评论围攻、恶意剪辑、单个平台扩散 | 先平台举报、申诉、留证 |
| 出现手机号、住址、学校、单位、家人信息 | 平台处理和报警准备同时进行 |
| 出现威胁、敲诈、跟踪、堵门、线下试探 | 先尽快报警，同时继续留证 |
| 未成年人信息被公开，或家人已被直接骚扰 | 不要拖，尽快升级处理 |

## 什么情况先平台举报

1. 内容主要还停留在线上公开扩散，没有明显线下接触迹象。
2. 你需要先让原帖、评论区、搬运内容尽快下线，减少继续外溢。
3. 对方账号、帖子、评论、搜索联想、私信等都还在平台内可定位。
4. 你已经能较清楚说明哪条内容违规、哪类信息被公开、希望平台处理什么。

## 什么情况尽快报警

1. 已出现明确的人身威胁、跟踪、堵门、上门、敲诈、线下围堵。
2. 公开内容里已经包含住址、学校、工作地点、家人联系方式、未成年人信息。
3. 对方开始联系家人、单位、老师、同事或邻居，风险已明显从线上转到线下。
4. 你已经接到持续骚扰电话、短信轰炸，或有人试图套取验证码、地址等进一步信息。

## 报警前先准备什么

1. 最核心的几份证据：原帖截图、评论截图、账号主页、原始链接、录屏和关键时间点。
2. 一份简短时间线：什么时候发现、公开了什么、传播到哪里、你已经做了什么。
3. 如果已有骚扰电话、短信、私信、线下接触，也一起整理时间、号码、账号和内容。
4. 能说明现实风险的材料优先，例如住址、学校、家人信息、威胁措辞，而不是只带情绪表达。

## 举报和报警可以并行

不要把它理解成“二选一”。

- **平台举报**的目标是先压扩散、先下线、先封禁。
- **报警**的目标是应对已经出现的人身和线下风险。

很多时候最稳妥的做法是：你或朋友继续在线留证、申诉、举报；同时把核心材料整理出来，尽快报警。

## 常见误区

1. **误区一：等更严重了再说。** 真正危险的是“已经出现线下信号，你还当成普通舆论冲突”。
2. **误区二：只盯着删帖。** 删帖能减少扩散，但不能替代对现实风险的判断。
3. **误区三：觉得材料不够完整就不敢行动。** 先把最关键的材料带上，后续还能补。
4. **误区四：把报警当成情绪宣泄。** 你需要的是事实、时间线和风险点，不是长篇背景描述。
